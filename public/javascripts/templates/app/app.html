Getting Started

How to build the <%= model.get('name') %> app with awesomebox

Building an app with awesomebox is incredibly straightforward. Just drop your files in the app directory and start the server!

Quick Start

Now that you have created a new app, let's walk through a simple hello world.

Open the code

First, let's open up our code directory by clicking the button in the top right of this window.

Create the homepage

Now, create a file and name it index.html. Let's add some simple HTML to this file: 

<!DOCTYPE html>
<html>
<head>
  <title><%= model.get('name') %></title>
</head>
<body>
  Hello World!
</body>
</html>

Run the local server

Now run the app by clicking the button, so that we can see what this looks like.

Ship the app!

Click the button to ship the app up to the awesomebox.es server!
